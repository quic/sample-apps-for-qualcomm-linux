
# Text-to-Image
## Call Flow
1. User requests image generation with text prompt.
2. The Web‑UI forwards the prompt to the Text‑to‑Image container through REST API.
3. The Text‑to‑Image service generates one or more images.
4. The Web‑UI returns the generated images back to the web page for rendering.
5. Generated image is rendered on the UI
